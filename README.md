Problem Statement:
This project focuses on building an NLP pipeline that can generate concise summaries from lengthy documents while preserving the most relevant information. The problem falls under text generation and information condensation, where the model identifies important sentences or patterns to produce meaningful summaries.

Data Type:
Unstructured long-form text data such as articles, reports, or paragraphs requiring preprocessing, tokenization, and vectorization.

Dataset Used:
A collection of textual documents paired with summaries or long passages used for extractive summarization techniques. The dataset enables the model to learn how to reduce content length while retaining semantic meaning.
